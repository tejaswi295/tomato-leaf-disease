Literature Survey

Key Research Foundations

2016

Mohanty et al.

Deep learning for plant disease detection using PlantVillage. Achieved 99.35% on lab images but poor field generalization.

2016

Radford et al. (DCGAN)

Unsupervised representation learning with DCGANs. Established architectural guidelines for stable GAN training.

2019

Tan & Le (EfficientNet)

Compound scaling of CNNs. EfficientNet-B0 achieves 77.1% top-1 ImageNet accuracy with only 5.3M parameters.

2020

Zhang et al. (MixUp)

Empirical risk minimization via convex combinations. Reduces memorization and overfitting significantly.

2020

Selvaraju et al. (Grad-CAM)

Visual explanations from deep networks. Enables explainable AI for model trust & debugging.

2021

Abbas et al.

GAN-based augmentation for tomato disease classification. Synthetic data improved minority class recall by 12%.

Literature Survey

Identified Research Gaps

⚠ Data Leakage

Most studies do not account for near-duplicate images leaking across train/test splits, inflating accuracy metrics.

⚠ Poor Generalization

Models trained on controlled lab images fail on real-world field photos due to lack of robust augmentation.

⚠ No Explainability

Black-box CNN classifiers lack visual explanations, reducing trust among agronomists and farmers.

⚠ No End-to-End System

Research stops at model accuracy; no deployable web interface for real-time field diagnosis.

Model Development — Software

DCGAN Architecture

Generator

Input: 100-dim noise vector z ~ N(0,1)

Layer 1: Linear → 512×8×8

Layer 2: ConvT 512→256, 4×4, stride 2

Layer 3: ConvT 256→128, 4×4, stride 2

Layer 4: ConvT 128→64, 4×4, stride 2

Layer 5: ConvT 64→32, 4×4, stride 2

Output: ConvT 32→3, Tanh → 128×128 RGB

BatchNorm + ReLU after each layer

Discriminator

Input: 3×128×128 RGB image

Layer 1: Conv 3→32, LeakyReLU(0.2)

Layer 2: Conv 32→64, BN + LeakyReLU

Layer 3: Conv 64→128, BN + LeakyReLU

Layer 4: Conv 128→256, BN + LeakyReLU

Layer 5: Conv 256→512, BN + LeakyReLU

Output: FC 512×4×4 → 128 → 1, Sigmoid

Dropout(0.3) before final layer

Loss: BCEOptimizer: Adam (lr=0.0002, β1=0.5)Epochs: 100Batch: 64

Model Development — Software

Classifier: EfficientNet-B0

Backbone

EfficientNet-B0 pretrained on ImageNet (5.3M params). Base layers frozen; final 2 conv stages unfrozen for fine-tuning.

Classification Head

Linear(1280 → 512) → BatchNorm → ReLU → Dropout(0.3) → Linear(512 → 10)

Anti-Overfitting

MixUp(α=0.2), Label Smoothing(0.1), RandomErasing, Gaussian Noise, Gradient Clipping(1.0)

Training

AdamW (lr=5e-4, wd=1e-4), CosineAnnealing LR, Early Stopping (patience=8), Weighted CrossEntropy

Conclusions

Key Takeaways

1

Robust Classification

EfficientNet-B0 with transfer learning achieves high accuracy across all 10 tomato disease classes while maintaining generalization through aggressive augmentation and anti-overfitting techniques.

2

GAN-Based Data Augmentation

DCGAN generates realistic synthetic diseased leaf images, addressing class imbalance and improving minority class recall without manual data collection.

3

Explainable AI

Grad-CAM heatmaps provide visual evidence for predictions, enabling farmers and pathologists to trust and verify the model's decision-making process.

4

End-to-End Deployment

Complete full-stack system (FastAPI + React) enables real-time disease diagnosis from any device with a web browser — bridging the gap from research to practical field use.

Future Work

  • Real-time camera feed disease detection for in-field use
  • Expand to multi-crop disease detection beyond tomato
